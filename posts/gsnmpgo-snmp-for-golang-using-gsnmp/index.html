gsnmpgo - SNMP for Golang using gsnmp

Sun Feb 10, 2013

Update 8/Mar/13

Work on gsnmpgo has halted. Making the gsnmp C library multithreaded was proving too time consuming. Use http://github.com/soniah/gosnmp instead.

Previously…

I recently released gsnmpgo - a Go/CGo snmp library using gsnmp. Pull requests welcome!

From the gsnmpgo documentation, here’s an example of usage:

[sourcecode language=“cpp”] // do an snmp get; RFC 4088 is used for uris uri := snmp://public@192.168.1.10//(1.3.6.1.2.1.1.1.0) params := gsnmpgo.NewDefaultParams(uri) results, err := gsnmpgo.Query(params) if err != nil { fmt.Println(err) os.Exit(1) }

// check your results gsnmpgo.Dump(results)

// turn on debugging gsnmpgo.Debug = true [/sourcecode]
