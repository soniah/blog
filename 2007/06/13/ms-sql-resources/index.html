Useful addins

  • AnkhSVN – Subversion source code control plugin for Visual Studio
  • Log PI and Log Explorer – both allow manipulation of SQL Server logs eg remove a single transaction
  • PromptSQL – provides SQL Intellisense style word completion for various Microsoft SQL editors, and UltraEdit
  • ReSharper – features such as intelligent coding assistance, on-the-fly error highlighting and quick error correction, support for code refactoring
  • Reflector – allows you to view, navigate, search, decompile and analyze .NET assemblies. Other tools also on the same site.
  • Quest Spotlight – “Spotlight on SQL Server Enterprise provides an agent-less easy-to-use database issue discovery solution that enables you to identify and resolve SQL Server performance problems within your SQL Server environment.“
