    Setup Arduino on Debian

    26 Feb 2014
    Some notes on setting up Arduino on Debian. Add yourself to the required groups - dialout and tty. [sourcecode lang=“shell”] sudo adduser $USER dialout sudo adduser $USER tty [/sourcecode] Confirm the port /dev/ttyUSB0 is being used by Arduino - tail messages then plug in the Arduino. [sourcecode lang=“shell”] sudo tail -f /var/log/messages you should see something like: FTDI USB Serial Device converter now attached to ttyUSB0 out of interest, inspect permissions on port ls -al /dev/ttyUSB0 [/sourcecode] Logout and login (or run a login shell).
